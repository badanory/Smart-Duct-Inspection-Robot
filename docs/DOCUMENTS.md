# Documents Index

이 폴더는 HappyCircuit 프로젝트의 포트폴리오 증빙 자료를 분류해 둔 공간입니다.

## Key Entry Points

- `../README.md`: 포트폴리오형 프로젝트 개요
- `architecture.md`: Mermaid 기준 시스템 아키텍처
- `notes/legacy_deployment_notes.md`: 당시 운영 메모

## Folders

### awards

프로젝트 수상 상장 자료입니다.

- 한이음 드림업 장려상
- KIPS 우수논문상 동상
- 캡스톤디자인 최우수상
- Withus 우수동아리상

### papers

논문 본문 또는 논문 관련 자료입니다.

- `AI 기반 덕트 자율주행 로봇의 설계와 케이블 결함 탐지 성능 비교.pdf`
- `덕트 환경 내 이미지 데이터 송신을 위한 프로토콜 연구.pdf`
- `websocket_latency_ws_rx_metrics.csv`

세부 설명은 `papers/README.md`에 정리했습니다.

### reports

최종보고서, 결과보고서, 제작설계서, 주간보고서 등 프로젝트 진행 문서입니다.

### presentations

발표 자료, 포트폴리오 PDF, 판넬 자료입니다.

### media

사진과 영상 링크 메모가 들어 있습니다.

- `demo_video_links.md`
- 현장/증빙 사진 2장

### notes

실제 운영 당시 메모와 레거시 서버 코드 스냅샷입니다.

- `legacy_deployment_notes.md`
- `legacy_rtsp_server_app.py.txt`
